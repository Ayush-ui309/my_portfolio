# Ayush Kumar Singh - Developer Portfolio 🚀

A stunning, interactive, and highly responsive personal portfolio website built to showcase my projects, skills, and professional experience. Designed with a deep, glassmorphic "Neon Dark" aesthetic utilizing cutting-edge web technologies.

## 🌟 Key Features
- **Immersive UI/UX**: Deep dark mode with vibrant blue, purple, and emerald mesh gradient neon highlights.
- **Glassmorphism**: Beautiful translucent cards with backdrop blurs and subtle interactive borders.
- **Dynamic Animations**: Powered by Framer Motion and Lenis for smooth scrolling, viewport-triggered reveals, and 3D geometric container rotations.
- **Embedded Resume Viewer**: Custom-built `/resume` route featuring an embedded HTML/PDF viewer with a direct download action.
- **Animated Metrics**: Live counting integers tracking lines of code, solved DSA problems, and hackathon participations natively.

## 🛠️ Technologies Used
- **Frontend**: React.js, Vite
- **Styling**: Tailwind CSS, PostCSS
- **Animations**: Framer Motion, Lenis (Smooth Scroll)
- **Icons**: Lucide React, React Icons (Official SVG logos)
- **Routing**: React Router DOM

## 🚀 Getting Started Locally

To run this portfolio on your local machine, follow these steps:

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Ayush-ui309/my_portfolio.git
   cd my_portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **Build for production:**
   ```bash
   npm run build
   ```
